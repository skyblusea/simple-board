## 🎯 구현 기능

1️⃣ 사용자 회원가입
2️⃣ 로그인
3️⃣ 로그인한 사용자 정보 표시
4️⃣ 게시글 기능 (CRUD)

<br/>

## 🚀 시작하기

### 1️⃣ 의존성 설치

```bash
pnpm install
```

### 2️⃣ 개발 서버 실행

```bash
pnpm dev
```

브라우저에서 `http://localhost:5173` 으로 접속합니다.
